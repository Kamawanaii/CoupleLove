# Telegram Couple App

Личное Telegram Mini App для двоих.

Что внутри:

- вход только по коду `291224`
- код вводится один раз на профиль Telegram
- максимум 2 подтверждённых пользователя
- случайный вопрос каждые 24 часа
- возврат к прошлым вопросам за валюту
- недельный бонус валюты за 5 ответов из 7
- блог и воспоминания с фото
- точки на карте мира по координатам
- тематические вопросы с процентом совпадения
- месячные итоги

## Запуск

1. Скопируйте `.env.example` в `.env`
2. Укажите данные нового бота и публичный HTTPS URL
3. Запустите:

```bash
npm run dev
```

4. Откройте `http://localhost:3001`
